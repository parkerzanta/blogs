Cara Install Composer Di Termux

Download as .zip Download as .tar.gz View on GitHub

Bahan bahan yang perlu di siapkan

  • Termux
  • Kuota
    • jika semua bahan telah terpenuhi lanjut kita install Composer nya, tapi sebelumnya termux kalian harus sudah terinstall cURL PHP, jika belum terinstall maka silahkan install dengan cara berikut.

Lalu ketikan perintah di bawah pkg install php pkg install curl

jika cURL PHP telah terinstall, maka langkah selanjutnya adalah menginstall Composernya.

Ketikan curl -sS https://getcomposer.org/installer | php -- --install-dir=/data/data/com.termux/files/usr/bin --filename=composer

jika Composer telah terinstall, maka tampilannya akan seperti ini

lalu silahkan cek dengan mengetikkan perintah composer lalu jika ingin mengecek berapa versi Composer yang baru saja diinstall, ketikkan perintah composer -v
